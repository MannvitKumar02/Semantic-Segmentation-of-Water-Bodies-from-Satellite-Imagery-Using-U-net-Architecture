# Semantic-Segmentation-of-Water-Bodies-from-Satellite-Imagery-Using-U-net-Architecture

Deep learning project for water body segmentation using U-Net.

## Files

- `water_segmentation_pipeline.ipynb` — pipeline walkthrough

- Python 3.x
- TensorFlow / PyTorch
- OpenCV, NumPy, Matplotlib
